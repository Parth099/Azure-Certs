# Running the Python Code Here

-   (1) Create Venv

```shell
python3 -m venv ai102
```

-   (2) Activate env (_may be different on your machine_)

```shell
source ./ai102/bin/activate
```

-   (3) Install Libs

```shell
pip install flask requests python-dotenv pylint matplotlib pillow
pip install --upgrade numpy
pip install azure-cli
```

-   (4) Labs are [here](https://microsoftlearning.github.io/AI-102-AIEngineer/).
